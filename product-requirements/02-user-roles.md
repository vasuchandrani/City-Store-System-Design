## User Roles

### Customer

Customers use CityShop to:

- Discover local shops
- Browse products
- Search products
- Place orders
- Track deliveries
- Communicate with shopkeepers

---

### Shopkeeper

Shopkeepers use CityShop to:

- Create shop profile
- Manage products
- Manage inventory
- Receive orders
- Run advertisements
- Expand customer reach

---

## Delivery Partner

Delivery partners use CityShop to:

- Accept delivery requests
- Pick up orders
- Deliver products
- Track earnings
